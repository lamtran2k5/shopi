## How to run 
- docker compose up --build
- docker compose exec php-fpm sh  
     composer install
- docker compose exec php-fpm composer require laravel/breeze
- docker compose exec php-fpm composer require laravel/sanctum
- docker compose exec php-fpm php artisan migrate --seed

## About project 
This is an e-commerce website developed using PHP Laravel, featuring both a shopping system for customers and an admin management dashboard for store owners.
